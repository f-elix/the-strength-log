# The Strength Log

A simple strength training log PWA, perfect for tracking workouts.

## Technologies and packages used

### Client side

-   Vue / Vue-router
-   XState
-   Workbox / vue-pwa-plugin
-   Vue material
-   Vuedraggable
-   Vee-validate

### Server side

-   Node.js
-   Graphql / Apollo-server
-   MongoDB / Mongoose
-   JWT
